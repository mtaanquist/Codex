# Keyboard shortcuts

Codex keeps to a small set of shortcuts. On a Mac, use Cmd where Ctrl is shown.

## Anywhere

- **Ctrl+K**: open the command palette to jump to any universe, story, scene, or
  entity, or run a command (new scene, switch view, focus mode, and more).

## In the editor

- **Ctrl+B**: bold the selection.
- **Ctrl+I**: italicise the selection.
- **Ctrl+F**: find and replace within the scene.
- **Ctrl+Alt+M**: mark the selected text as a TODO. A line that begins with
  `TODO:` becomes one on its own, no shortcut needed.
- **Ctrl+Z** and **Ctrl+Shift+Z**: undo and redo.
- **Ctrl+J**: ask the Assistant to continue the passage at the cursor (when the
  Assistant is set up). Tab accepts the grey suggestion; Esc dismisses it.

In the continuous "read the whole story" view, the up and down arrows carry the
cursor across scene boundaries, so you can move through the book as one document.

## Finding more

Most actions also live in the command palette (Ctrl+K) and in the menus and
buttons around the editor, so you do not have to memorise shortcuts to reach
them.
